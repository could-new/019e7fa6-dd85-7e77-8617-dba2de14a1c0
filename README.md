# Motorista Pro BR

Um aplicativo de mobilidade desenvolvido especialmente para beneficiar os motoristas brasileiros. O aplicativo foca em repasse integral (zero impostos embutidos pelo app) e inclui configurações avançadas para metas de ganhos.

## Funcionalidades Principais

*   **Painel do Motorista:** Uma visão clara e rápida das solicitações de corrida e status de disponibilidade.
*   **Ganhos Sem Impostos:** Um sistema focado na transparência, mostrando o valor real sem descontos abusivos.
*   **Configurações Avançadas:** Opções personalizadas para o motorista, incluindo a definição de metas de bônus e salário mínimo (iniciando com um bônus padrão a partir de R$ 200).
*   **Design Responsivo:** Funciona perfeitamente em dispositivos móveis, tablets e telas de diferentes tamanhos.

## Tecnologias Utilizadas

*   Flutter (Cross-platform para Android, iOS, Web e Desktop)
*   Material Design 3

## Como Executar

1.  Certifique-se de ter o Flutter instalado e configurado em seu ambiente.
2.  Clone o repositório.
3.  Execute `flutter pub get` para baixar as dependências.
4.  Execute `flutter run` para iniciar o aplicativo no seu dispositivo ou emulador preferido.
